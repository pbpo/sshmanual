Unified Lab Environment

Directory Overlay (Persistence)

You are inside a container, but your changes are saved. You can run apt install to install packages, and they will persist next time you log in.

Instance Join (Auto-Resume)

Login automatically joins your existing background instance. Your Tmux/Byobu sessions are shared across all SSH connections.

Fakeroot Environment

You have root privileges inside the container. Use sudo or just run commands directly. Note: You cannot modify the host kernel/drivers.
